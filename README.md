## Deploy Steps (Browser only, no PC needed)

### Step 1: GitHub
1. github.com → New repository → Name: `flipkart-clone`
2. Upload all files from this zip (keep folder structure)
3. Commit

### Step 2: Vercel
1. vercel.com → Sign up with GitHub
2. New Project → select `flipkart-clone`
3. Framework: Next.js (auto detected)
4. Deploy → Live in 2 mins ✓
